# Contributing to codemirror-flutter

Thanks for your interest in contributing!

## Setup

```bash
task bootstrap
```

## Running Tests

```bash
task test
```

## Pull Request Guidelines

1. Open an [issue](https://github.com/pocketbase/codemirror-flutter/issues) first to discuss significant changes
2. Run `task check` before submitting to ensure code passes all checks
3. Keep PRs focused on a single change
4. Include tests for new functionality

## Questions?

Open an issue for discussion.
