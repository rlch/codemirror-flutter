# Changelog

## 0.0.1

- Initial release
- Pure Dart/Flutter port of CodeMirror 6
- EditorState, transactions, facets, extensions
- Syntax highlighting for JavaScript/TypeScript/JSX
- Autocompletion with fuzzy matching
- Bracket matching and auto-closing
- Code folding
- Search and replace
- Multiple selections
- Line numbers and gutters
- LSP integration hooks (hover, diagnostics, go-to-definition)
